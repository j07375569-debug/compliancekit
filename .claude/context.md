# Domain Context

## What Compliance SaaS Does
Automates the process of getting certified for security and privacy frameworks. Without a tool like this, startups hire $50k+/year consultants or spend months doing it manually.

## Target Frameworks
- **SOC 2** — Most requested by enterprise buyers. Type I (point-in-time) and Type II (over period). Controls: access control, encryption, incident response, vendor management.
- **HIPAA** — Required for healthcare data. PHI protection, BAA agreements, breach notification.
- **GDPR** — EU data privacy. Consent management, data subject rights, DPO appointment, breach reporting within 72 hours.
